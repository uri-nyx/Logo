# Logo
My version of LOGO with Animation

Run Here: http://htmlpreview.github.io/?https://github.com/imlolman/Logo/blob/master/index.html

# Commands Supported

rt - right turn <br>
lt - left turn <br>
fd - forward <br>
bk - backward <br>
pu - pen up <br>
pd - pen down <br>
ht - hide turtle <br>
st - show turtle <br>
repeat - to repeat the commands. <br>

# Repeat syntax

` repeat loopCount [ commands ] `
